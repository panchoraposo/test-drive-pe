MCP Server — customer-service-mcp

El MCP Server expone tools para que OpenShift Lightspeed y agentes AI gestionen clientes vía protocolo MCP (SSE + JSON-RPC). Acá puedes ver su estado y las tools disponibles.

Tools disponibles

createCustomer

Crea un nuevo cliente en NeuralBank con datos de identificación, contacto y scoring inicial.

getCustomer

Obtiene información completa de un cliente por su ID único en el sistema.

getCustomerByIdentification

Busca un cliente por su número de identificación (RUT, DNI, pasaporte).

searchCustomers

Busca clientes con filtros: nombre, email, tipo, ciudad. Resultados paginados.

updateCustomer

Actualiza datos de un cliente existente (nombre, email, teléfono, ciudad).

getCreditScore

Consulta el score crediticio actual y evaluación de riesgo de un cliente.

calculateCreditScore

Recalcula el score crediticio basado en los datos financieros actuales.

updateRiskLevel

Actualiza el nivel de riesgo con justificación: Bajo, Medio, Alto, Muy Alto.

activateCustomer

Activa una cuenta de cliente para permitir transacciones.

deactivateCustomer

Desactiva una cuenta con motivo, impidiendo nuevas transacciones.

Arquitectura del flujo

Frontend → httpd proxy → Gateway (Istio + Kuadrant) → Backend API
Lightspeed/AI → MCP Gateway (Kuadrant) → MCP Server → Backend API

AuthPolicy: X-API-Key → Kuadrant Authorino → Allow/Deny
RateLimitPolicy: Kuadrant Limitador → Throttle
