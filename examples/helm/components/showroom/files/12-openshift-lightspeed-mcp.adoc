= 10. OpenShift Lightspeed & MCP Gateway

**OpenShift Lightspeed (OLS)** is an AI assistant integrated into the OpenShift web console. Combined with the **Red Hat Connectivity Link MCP gateway** (Technology Preview), it can execute cluster operations through AI-callable tools.
+
The MCP gateway operator is installed from the `redhat-operators` catalog (`preview` channel). For details, see the official documentation: `https://docs.redhat.com/en/documentation/red_hat_connectivity_link/1.3/html/installing_the_mcp_gateway/mcp-gateway-install`.

== Accessing OpenShift Lightspeed

. Log in to the **OpenShift Console** at `\https://console-openshift-console.{cluster_domain}`
. Click the **Lightspeed** icon in the console header (chat bubble icon).
. Ask questions about your cluster, workloads, or Kubernetes concepts.

== Before You Run Prompts

Two UI tips for a smoother experience:

- **Tool output vs chat**: when a tool runs, the detailed result shows in **Tool output**. The assistant message may be short on purpose.
- **3D model**: use the Tech Day icon (bottom-right) to open the interactive 3D view and relate prompts to MCP servers/tools.

== Workshop Prompt Walkthrough (using MCP tools)

Run the following prompts in the Lightspeed chat. When Lightspeed asks to run a tool/action, approve it.

. List pods in the pre-deployed stack:
* `List all pods in the neuralbank-stack namespace`
. List Argo CD Applications:
* `List all ArgoCD applications`
. Scale the frontend in your scaffolded namespace:
* `Scale the frontend deployment to 2 replicas in the {user_name}-neuralbank namespace`
. Create an HPA for the backend (explicit parameters help tool selection):
* `Create a HorizontalPodAutoscaler for deployment neuralbank-backend in the {user_name}-neuralbank namespace with min 1 max 3 cpu 70`
. Deploy Redis in your scaffolded namespace (choose one):
* `Deploy a Redis database in the {user_name}-neuralbank namespace`
* `Install Redis using Helm in the {user_name}-neuralbank namespace`

. Developer Hub integration (direct tools):
* `List all Software Templates available in Developer Hub`
* `Show me all registered components`
* `Register a new entity from https://gitea-gitea.{cluster_domain}/ws-{user_name}/ols-import-demo/raw/branch/main/template.yaml`
